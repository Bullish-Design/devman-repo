## Quick start

- If you use `direnv`, allow it:

```bash
direnv allow
```

- Or run devenv through devman:

```bash
devman run shell
```

## Handy commands

- Enter shell: `devman run shell`
- Run tests: `devman run test`
- Run the example CLI (if enabled): `devman run hello -- <name>`
